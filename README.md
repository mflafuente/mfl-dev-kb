# mfl-dev-kb

Base de conocimiento personal para proyectos de desarrollo y configuración de [opencode](https://opencode.ai).

```
mfl-dev-kb/
├── index.md       ← Punto de entrada
├── opencode/      ← Plugins, agentes y providers instalados
├── projects/      ← Notas y decisiones por proyecto
├── resources/     ← Documentos y assets preservados
└── skills/        ← Skills reutilizables
```

## Stack

- **Editor:** opencode + Gemini 2.0 Flash + modelos locales (Qwen vía Ollama)
- **Contenido:** Markdown, organizado como índice de conocimiento
- **Filosofía:** La KB referencia lo instalado en `~/.config/opencode/` — este repo es el índice.

→ Ver [`index.md`](index.md) para más detalle.
