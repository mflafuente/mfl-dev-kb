# mfl-dev-kb

Base de conocimiento personal para proyectos de desarrollo.

## Estructura

```
mfl-dev-kb/
├── index.md               ← este archivo
├── projects/              ← Notas, decisiones y aprendizajes por proyecto
├── resources/             ← Documentos, assets y elementos preservados
├── skills/                ← Skills reutilizables para opencode (referencias)
└── opencode/              ← Referencias a complementos instalados en opencode
```

## Filosofía

Esta KB es un **índice de conocimiento**. Los complementos reales (skills, plugins,
MCPs) viven en `~/.config/opencode/`. Acá dejamos:

- Referencia de lo que está instalado y para qué sirve
- Enlaces a descargas o documentación original
- Notas de por qué se instaló y cómo se usa
